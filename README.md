# rest-mongo-flask
Criar uma API REST em Python, com Flask e MongoDB

## Setup

### Dependencies 
1. Windows+WSL+Ubuntu22.04 LTS or Ubuntu22.04
2. Python 3.11
   1. pyenv
   2. pipenv or pip
3. Docker
4. Docker Compose v2
